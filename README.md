to install all of the requirements to run the code please run:
$ pip install -r requirements.txt

to run the program type this into the terminal:
$ python political_quiz.py issues_all.txt

database folder contains all of the politicians information from their websites directly

the politicians.txt is the list the system reads to get the politicians names

issues.txt is where all of the keywords for each issues and the cooresponding key words are
